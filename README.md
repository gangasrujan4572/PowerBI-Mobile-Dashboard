# 📊 Power BI Mobile Dashboard

A responsive Power BI dashboard designed for mobile devices to visualize business insights using interactive charts, KPIs, and filters.

## Features

- 📱 Mobile optimized layout
- 📈 Interactive visualizations
- 📊 KPI Cards
- 📅 Date filtering
- 🔍 Drill-through capability
- 📌 Responsive dashboard

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Excel / CSV Dataset
